Use "< p >" to form paragraphs. Paragraphs are an example of block element types. Block elements can only be contained within other block elements. Another example of a block element type is the unordered list.

Block Elements

  1. Paragraphs
  2. Lists (unordered and ordered)
  3. Headings
  4. Sections
  5. Long quotes

The other type of elements are inline elements. These can be contained within other inline elements or block elements.

Inline Elements

  • Links
  • Short quotes
  • Abbreviations
  • Font styles (bold, italics)
