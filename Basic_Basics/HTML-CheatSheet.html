HTML Cheat Sheet

This text is contained under the element: "< p >" to form paragraphs. Paragraphs are an example of block element types. Block elements can only be contained within other block elements. Another example of a block element type is the unordered list.

Block Elements

  1. Paragraphs: "p"
  2. Lists (unordered: "ul" and ordered: "ol")
  3. Headings: "h#"
  4. Sections: "section"
  5. Long quotes: "blockquote"

The other type of elements are inline elements. These can be contained within other inline elements or block elements.

Inline Elements

  • Links: "a"
  • Short quotes: "q"
  • Abbreviations: "abbr"
  • Font styles (bold: "strong", italics: "i" or "em")
